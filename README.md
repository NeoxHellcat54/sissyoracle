# The Great Oracle of Sissy Sluts - PWA Starter

A plain HTML/CSS/JS progressive web app for generating decisive adult fantasy/session ideas from selected tags.

## Run locally

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Files

- `index.html` - app shell
- `styles.css` - styling
- `app.js` - tag lists, generator logic, local storage
- `manifest.json` - PWA manifest
- `sw.js` - offline cache/service worker

## Notes

This version intentionally makes the generated session specific and directive in tone: concrete outfit, concrete activity, concrete kink/fantasy layer, and concrete exposure/photo layer when selected.
